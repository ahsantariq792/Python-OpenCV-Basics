## RGB (Red, Green, Blue)

Steel grey

https://www.htmlcsscolor.com/hex/43464B


### Main

1. (0, 0, 0) = Black
2. (255, 255, 255) = White
3. (255, 0, 0) = Blue
4. (0, 255, 0) = Green
5. (0, 0, 255) = Red

### Combinations

1. (255, 255, 0) = Blue + Green = Cyan
2. (255, 0, 255) = Blue + Red = Magenta
3. (0, 255, 255) = Green + Red = Yellow
